PreReq:
Python 3 installed
(https://www.python.org/downloads/)
Pandas installed
(pip install pandas)
openpyxl installed
(pip install openpyxl)


How to use:
1. Make sure prereqs are met (can use pip install -r req.txt)
2. Paste ticket_export.csv into folder (ensuring the name is the same)
3. Run python script (python ticketcleaner.py or use run.bat)
4. Extract cleaned data from currentcleantickets.xlsx and paste into ticketpiviot.xlxs in the "Tickets go here" sheet
5. Right click piviot tables and hit refresh and you should be all set!
